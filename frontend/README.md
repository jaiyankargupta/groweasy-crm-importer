# GrowEasy CRM — Frontend UI

This is the Next.js frontend application for the **GrowEasy CRM Lead Importer**, built with React 19, App Router, and a clean **Enterprise SaaS** design system.

## Features

- **Groq LPU Integration**: Choose between Llama 3.3 70B, Llama 3.1 8B, Mixtral 8x7B, and Gemma 2 9B for ultra-fast lead extraction.
- **3-Step Ingestion Flow**:
  1. **Upload**: Drag-and-drop CSV upload or 1-click sample dataset selection.
  2. **Preview**: Verify raw column data and structure before running AI.
  3. **Results**: Review successfully mapped CRM leads and skipped invalid rows with 1-click JSON/CSV exports.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.
